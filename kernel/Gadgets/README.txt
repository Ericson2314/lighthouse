This directory and the lib/ subdirectory contains a port of the
Gadgets system, developed at York University by Rob Noble and Colin
Runciman. It was originally developed for use with the Component Gofer
interpreter. In 2004, the system was ported to Concurrent Haskell by
Thomas Hallgren.
